# Surftorial — 브랜드 가이드라인

> 서핑 온라인 강좌 플랫폼 | Brand Guidelines v1.0

---

## 1. 브랜드 개요

### 미션
파도 위에서의 경험을 온라인으로. Surftorial은 초보자부터 중급 서퍼까지, 체계적인 서핑 강좌를 언제 어디서나 제공합니다.

### 비전
서핑 교육의 디지털 표준 — 현장 중심의 서핑 레슨을 온라인으로 확장하여, 더 많은 사람이 파도와 만날 수 있도록 합니다.

### 브랜드 키워드
- **해양 (Oceanic)** — 바다, 파도, 자연
- **성장 (Progression)** — 레벨업, 단계적 학습
- **접근성 (Accessible)** — 누구나 쉽게, 언제 어디서나
- **따뜻함 (Warmth)** — 커뮤니티, 코칭, 격려

## 2. 로고 컨셉

### 로고 마크 (Symbol Mark)
파도의 곡선과 재생 버튼(▶)이 결합된 형태. 서핑의 역동성과 온라인 강좌의 '재생/학습'을 동시에 상징.

```
    ╭─────╮
   ╱       ╲        ← 파도 곡선 (wave curve)
  ╱    ▶    ╲       ← 재생 삼각형 내장
 ╱           ╲
╱─────────────╲     ← 수평선 (horizon line)
```

### 로고타입 (Wordmark)
- **서비스명**: Surftorial
- **한글명**: 서프토리얼
- **조합**: 심볼 마크(좌) + 워드마크(우) 가로 배치
- **단독 사용**: 심볼 마크만 앱 아이콘, 파비콘, 로딩 스피너에 사용

### 로고 안전 영역
- 심볼 마크 주변 최소 여백: 로고 높이의 50%
- 워드마크 주변 최소 여백: 텍스트 x-height의 100%

### 로고 변형
| 변형 | 배경 | 사용 처 |
|------|------|---------|
| 풀컬러 (Primary) | 밝은 배경 | 웹사이트, 명함, SNS |
| 리버스 (Reversed) | 어두운 배경 | 영상 워터마크, 다크모드 |
| 모노블랙 | 흰색 배경 | 인쇄물, 팩스 |
| 모노화이트 | 어두운 배경 | 실루엣 사용 |

## 4. 타이포그래피

### 한글 본문: Pretendard
- **용도**: 본문, UI 텍스트, 버튼, 폼 레이블
- **이유**: Apple 시스템 폰트와 호환되는 오픈소스 한글 산스서리프. 가독성과 호환성 균형.

### 영문/디스플레이: Inter
- **용도**: 영문 헤딩, 숫자, 가격, 브랜드 네임 "Surftorial"
- **이유**: 웹 최적화 가변 폰트, 숫자 표기 우수, 다국어 호환

### 타이포그래픽 스케일
| 레벨 | 폰트 | Weight | Size | Line-height | 용도 |
|------|------|--------|------|-------------|------|
| Display | Inter | 800 | 48px / 3rem | 1.2 | 랜딩 히어로 |
| H1 | Inter + Pretendard | 700 | 36px / 2.25rem | 1.3 | 페이지 타이틀 |
| H2 | Inter + Pretendard | 700 | 30px / 1.875rem | 1.35 | 섹션 타이틀 |
| H3 | Inter + Pretendard | 600 | 24px / 1.5rem | 1.4 | 카드 타이틀 |
| H4 | Inter + Pretendard | 600 | 20px / 1.25rem | 1.4 | 소제목 |
| Body LG | Pretendard | 400 | 18px / 1.125rem | 1.7 | 리드 문단 |
| Body | Pretendard | 400 | 16px / 1rem | 1.7 | 본문 |
| Body SM | Pretendard | 400 | 14px / 0.875rem | 1.6 | 캡션, 메타 |
| Caption | Pretendard | 500 | 12px / 0.75rem | 1.5 | 라벨, 배지 |

### 타이포그래피 규칙
- 한영 혼용 시 한글과 영문 사이 스페이스 1칸 (자간 조정은 불필요)
- 숫자는 Inter (tabular-nums) 사용
- 가격 표기: `₩29,000` 형식 (Inter weight 700)
- 모바일에서 Display는 36px로 축소 (clamp 사용 권장)

## 5. 일러스트레이션 & 이미지 가이드

### 사진 스타일
- **서핑 액션 샷**: 역동적인 파도 위 서핑 장면 (저녁 골든아워 선호)
- **라이프스타일**: 해변 준비, 스트레칭, 웃는 서퍼 — 따뜻하고 긍정적
- **코칭/교육**: 인스트럭터와 학생, 지상 연습 — 전문성과 접근성
- **커뮤니티**: 그룹샷, 하이파이브, 비치 파티 — 소속감

### 사진 필터
- 색온도: 약간 따뜻하게 (+100K)
- 채도: 자연 유지 (과포화 금지)
- 대비: 약간 높임 (파도 질감 강조)
- 보네티: 바다색 강조 (어안 효과 금지)

### 아이콘 스타일
- 라인 아이콘 (2px stroke, rounded cap)
- 24×24 뷰박스, 20×20 실제 영역
- 시맨틱 컬러 사용 (Ocean Main 기본, Sunset Orange 강조)
- 아이콘 세트: 파도, 서핑보드, 썬글라스, 썬크림, 나침반, 재생, 체크, 별

## 6. 보이스 & 톤 (Voice & Tone)

### 브랜드 톤
| 속성 | 설명 | 예시 |
|------|------|------|
| 친근 | 격식 없이 편안하게 | "파도 위에서 만나요!" (X) "바다가 부를 때, 시작하세요." (O) |
| 전문 | 서핑 지식은 정확하고 신뢰할 수 있게 | "팝업 요령 3가지" (O) |
| 격려 | 시작하는 사람에게 용기 | "처음이라 괜찮아요, 다 여기서부터 시작했으니까" |
| 솔직 | 과장 없이 사실 전달 | "일주일이면 마스터!" (X) "꾸준히 연습하면 확실히 달라집니다" (O) |

### 카피 라이팅 가이드
- **높임말 사용**: 해요체 (〜해요), 기본. 안내 문구는 합쇼체 (〜합니다)
- **영문 혼용**: 서핑 용어는 영문 그대로 (popup, takeoff, bottom turn)
- **숫자 표기**: Arabic 숫자, 콤마 구분 (10,000원, 25개 강좌)
- **CTA 버튼**: 동사 + 혜택 ("수강 시작하기", "무료로 둘러보기")
- **금지 어휘**: "완벽", "최고", "100%", 보장성 과장 표현

```
🌊 바다의 색  → 깊은 Navy → 밝은 Sky Blue 그라데이션
🌅 일출의 온기  → Sunset Orange 액센트
🏖️ 모래의 편안함 → Warm Neutral 배경
🏄 역동적인 선 → 파도 곡선, 보드 실루엣
▶️ 학습의 시작  → 재생 삼각형, 프로그레스 바
```
