## 3. 마케팅 믹스 (4P)

### Product (제품)
- **핵심**: 서핑 온라인 강좌 플랫폼 (입문~심화 레벨별 커리큘럼)
- **차별화**: 한국어, 한국 서핑 문화 특화, 코치 영상 피드백
- **확장**: B2B 스쿨 파트너십, 모바일 앱 (M+5)

### Price (가격)
- 스타터 ₩29,000/월, 프리미엄 ₩49,000/월 (프리미엄 진입)
- 연간 결제 20% 할인
- 단건 코스 구매 ₩19,000~₩39,000
- 오프라인 1회 레슨의 1/3~1/7 가격 포지셔닝
- (상세: pricing-policy.md 참조)

### Place (유통)
- **1차**: 웹사이트 (surftorial.com) — SEO/SEM 유입
- **2차**: 모바일 앱 (iOS/Android) — M+5 출시
- **3차**: 제휴 스쿨 오프라인 연계 — 현장에서 QR/안내

### Promotion (촉진)
- **프리런칭**: 콘텐츠 마케팅 (SEO, SNS) 중심, 제로 마케팅 예산
- **런칭**: 퍼포먼스 광고 ₩1.5M + PR + 제휴
- **런칭 후**: 콘텐츠 마케팅 + 레퍼럴 + B2B 네트워크
- (상세: launch-roadmap.md 참조)

## 5. B2B 파트너십 전략

### 5.1 서핑 스쿨 제휴 모델

| 항목 | 상세 |
|------|------|
| **제휴 형태** | 수익 공유 (Surftorial 30% : 스쿨 70%) |
| **스쿨 제공** | 강좌 콘텐츠, 코치 프로필, 오프라인 연계 |
| **Surftorial 제공** | 플랫폼, 결제, CRM, 마케팅 |
| **온보딩** | 1:1 세팅 지원 (영상 업로드, 스쿨 페이지 제작) |
| **최소 계약** | 3개월 |

### 5.2 스쿨 제휴 타겟 (초기 5곳)

| 지역 | 스쿨 유형 | 예상 파트너 수 | 우선순위 |
|------|-----------|---------------|----------|
| 양양 | 대형 서핑 스쿨 (서프홀릭급) | 2곳 | 최우선 |
| 제주 | 중형 서핑 스쿨 (서퍼스립컬급) | 2곳 | 우선 |
| 강릉/부산 | 소형 서핑 스쿨 | 1곳 | 이후 |

### 5.3 제휴 영업 프로세스

```
1. 리서치 → 스쿨 리스트업 (인스타/네이버 지도)
2. 아웃바운드 → 인스타 DM + 이메일 제안
3. 미팅 → 화상/대면 미팅 (15분 제안)
4. 파일럿 → 1개월 무료 체험
5. 정식 계약 → 수익 공유 계약 체결
```
