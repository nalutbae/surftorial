## 1. 런칭 타임라인 개요

```
프리런칭      베타           소프트런칭       공식런칭
(인지 구축)  (제품 검증)    (초기 유저)     (스케일)
───┬───────┬───────────────┬───────────────┬──────→
   M-3     M-2             M-1              M0
```

| 단계 | 기간 | 목표 | 핵심 KPI |
|------|------|------|----------|
| **프리런칭** | 런칭 전 3개월 | 인지 구축, 사전등록 확보 | 사전등록 500명, SNS 팔로워 1,000 |
| **베타** | 런칭 전 2개월 | 제품 검증, 초기 피드백 | 베타 유저 100명, NPS ≥ 40 |
| **소프트런칭** | 런칭 전 1개월 | 유료 전환 검증 | 유료 전환율 ≥ 5%, MRR ₩500K |
| **공식런칭** | M0 | 대규모 유저 획득 | MAU 1,000, MRR ₩3M |

- **사전등록 이메일 500개** 수집
- SNS 팔로워 1,000명 달성

## 3. 베타 단계 (M-2 ~ M-1)

### 목표
- **베타 유저 100명** 확보
- 핵심 기능 (강좌 시청, 진행률, 피드백) 사용성 검증
- **NPS ≥ 40** 달성

### 베타 운영 전략

| 항목 | 상세 |
|------|------|
| **모집** | 사전등록자 중 100명 선착순 + 서핑 커뮤니티 오픈 모집 |
| **기간** | 4주 (1주당 25명 그룹 론칭) |
| **혜택** | 프리미엄 1개월 무료 + 런칭 50% 할인 쿠폰 |
| **피드백** | 주간 설문 + 인앱 피드백 버튼 + 디스코드 채널 |
| **리워드** | 버그 리포트 건당 ₩5,000 크레딧 |

### 베타 검증 항목
| 검증 항목 | 기준 | 측정 방법 |
|-----------|------|-----------|
| 온보딩 완료율 | ≥ 70% | 가입→첫 강좌 시청 |
| 강좌 완주율 | ≥ 40% | 시작→완료 |
| 코치 피드백 만족도 | ≥ 4.0/5.0 | 피드백 후 설문 |
| 재방문율 (D7) | ≥ 30% | DAU/MAU |
| NPS | ≥ 40 | 베타 종료 설문 |

## 4. 소프트런칭 (M-1)

### 목표
- **유료 전환율 ≥ 5%** 검증
- **MRR ₩500,000** 달성
- 결제·환불 프로세스 실전 검증

### 소프트런칭 전략
- 베타 유저 대상 **얼리버드 50% 할인** (첫 결제 3개월)
- 네이버 블로그/인스타그램 팔로워 대상 **프라이빗 액세스**
- 제휴 서핑 스쿨 **1곳 파일럿** 시작
- 유료 전환 퍼널: 무료 체험 → 스타터 → 프리미엄 업셀

### 소프트런칭 일정
| 주차 | 액션 |
|------|-------|
| W1 | 베타→소프트런칭 전환, 얼리버드 프로모션 시작 |
| W2 | 첫 유료 전환 데이터 분석, 퍼널 병목 식별 |
| W3 | A/B 테스트 (가격, CTA, 온보딩) |
| W4 | 소프트런칭 결과 분석, 공식런칭 준비 |

## 8. 초기 유저 확보 전략 (User Acquisition Funnel)

```
인지(Awareness) → 관심(Interest) → 고려(Consideration) → 전환(Conversion) → 유지(Retention)

[SEO/블로그]     [인스타/유튜브]    [랜딩페이지]       [무료체험→유료]    [커뮤니티]
  10,000             5,000              1,000              150               100
  (도달)             (클릭)             (방문)             (결제)            (재구독)
```

### 퍼널 전환율 목표

| 단계 | 전환율 목표 | 전략 |
|------|------------|------|
| 인지→관심 | 50% (클릭률) | 매력적인 썸네일, 서핑 팁 타이틀 |
| 관심→고려 | 20% (랜딩 방문) | SEO 최적화, 광고 타겟팅 |
| 고려→전환 | 15% (무료체험 가입) | 명확한 CTA, 무료 체험 강조 |
| 전환→유료 | 5~8% (유료 전환) | 온보딩, 프리미엄 체험, 가치 증명 |
| 유료→유지 | 85% (월간 유지) | 콘텐츠 업데이트, 커뮤니티, 코치 연결 |
