## 1. 가격 철학

### 원칙
1. **현장 레슨 대비 1/10**: 1회 오프라인 레슨(₩60,000~₩200,000)의 1/10 가격으로 월간 접근 제공
2. **프리미엄 진입(Freemium)**: 무료 체험으로 퍼널 진입, 유료 전환 유도
3. **가치 기반 가격**: 코칭 피드백 포함 여부가 가격 차이의 핵심 축
4. **1인 운영 최소 비용**: 초기 고정비 최소화, 변동비 중심 구조

## 3. 할인 정책

### 연간 결제 할인
- 스타터: 월 ₩29,000 → 연간 ₩278,400 (**20% 할인**, 월 ₩23,200 상당)
- 프리미엄: 월 ₩49,000 → 연간 ₩470,400 (**20% 할인**, 월 ₩39,200 상당)

### 얼리버드 프로모션 (런칭 후 3개월)
- 스타터: 첫 달 ₩1,000 (97% 할인)
- 프리미엄: 첫 달 ₩9,900 (80% 할인)
- 연간 결제 시 추가 10% 할인 (총 28% 할인)

### 친구 초대 ( referral )
- 초대자: 다음 결제 ₩5,000 크레딧
- 피초대자: 첫 달 50% 할인
- 프리미엄 이상 구독자만 초대 크레딧 획득 가능

### 시즌 프로모션 (겨울 비시즌)
- 11~2월: 스타터/프리미엄 연간 결제 시 추가 10% 할인
- 목적: 비시즌에도 구독 유지 유도 (오프라인 레슨 불가 시기 = 온라인 수요 최대)

## 5. 결제 및 환불 정책

### 결제 수단
- 신용카드, 체크카드 (토스페이먼츠 정기결제)
- 카카오페이, 네이버페이
- 계좌이체 (연간 결제, B2B 계약)

### 환불 정책
| 상황 | 환불 |
|------|------|
| 무료 체험 기간 내 해지 | 전액 환불 (결제일로부터 7일) |
| 월 구독 해지 | 당월 말까지 이용, 익월 결제 안 함 |
| 연간 구독 해지 | 사용 월수 정액 계산 후 잔여 월 환불 (10% 위약금 공제) |
| 단건 구매 환불 | 결제일 7일 내 전액 환불, 7일 이후 환불 불가 |
| 코치 피드백 미제공 | 프리미엄 해당 회 피드백 1회 크레딧 또는 1개월 연장 |

| **Surf Hub** (글로벌) | 온라인 구독 | $9/월 (Basic), ~₩12,000 | 영어 중심, 한국어 없음 |
| **유튜브 서핑 튜토리얼** | 무료 | ₩0 | 파편화, 체계 없음, 한국어 적음 |
| **온라인 스포츠 코칭 플랫폼** (글로벌 평균) | 구독 | $15~30/월 | 스포츠 범용, 서핑 특화 없음 |
| **Surftorial** (본 서비스) | 구독 | ₩29,000/월 | 한국어, 서핑 특화, 코치 피드백 |

### 포지셔닝
- 오프라인 1회 레슨의 **1/3~1/7** 가격으로 **월간 무제한** 접근
- 글로벌 서핑 플랫폼(Surf Hub)보다 **한국어·한국 서핑 문화 특화**로 차별화
- 무료(유튜브)보다 **체계적 커리큘럼 + 코치 피드백**으로 차별화
