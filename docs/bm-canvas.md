## 1. 고객 세그먼트 (Customer Segments)

### B2C (개인 수강생)

| 세그먼트 | 페르소나 | 니즈 | 규모 추정 |
|----------|----------|------|-----------|
| **초보 서퍼** | 서핑 입문 희망자 (20~30대). 해변 리조트 방문객, SNS로 서핑에 관심 생긴 초보 | 기초 기술 체계 학습, 안전 가이드, 언제 어디서나 복습 | 한국 약 5만명/년 (서핑 체험객 기준) |

| **시즌 서퍼** | 여름/휴가철에만 서핑. 장비 없이 렌탈로 타는 캐주얼 서퍼 | 시즌 전 예습, 시즌 중 복습, 빠른 입문 | 한국 약 10만명/년 (해양스포츠 체험객 포함) |

| 세그먼트 | 페르소나 | 니즈 |

| **서핑 숙박/리조트** | 서핑 게스트하우스, 비치 리조트 | 게스트 부가 서비스, 패키지 상품화 |
| **지자체/스포츠회관** | 해양스포츠 진흥 지자체 담당자 | 시민 서핑 교육 프로그램 위탁 |

## 2. 가치 제안 (Value Proposition)

### B2C 개인 수강생

| 페인포인트 | 가치 제안 |
|-----------|----------|
| 현장 레슨은 1회 6~20만원, 반복 어려움 | 온라인으로 월 2.9만원에 무제한 수강 |
| 현장은 날씨·조건에 따라 취소 빈번 | 언제 어디서나 이론+영상 학습 가능 |
| 초보는 무엇부터 배워야 할지 막막 | 레벨별 커리큘럼 (입문→중급→심화) 체계적 제공 |
| 중급은 실력 정체, 피드백 부재 | 영상 업로드 → 코치 피드백 제공 (프리미엄) |
| 유튜브는 파편화, 체계 없음 | 단계별 코스 완주 구조, 진행률 추적 |

### B2B 파트너

| 페인포인트 | 가치 제안 |
|-----------|----------|
| 비시즌(겨울) 수익 급감 | 연중 온라인 수강으로 비시즌 수익 보완 |
| 1:1 레슨 스케일 불가 | 플랫폼으로 다수 학생 동시 관리 |
| 마케팅·콘텐츠 제작 역량 부족 | Surftorial이 콘텐츠·결제·CRM 제공 |

## 5. 수익 흐름 (Revenue Streams)

| 수익원 | 모델 | 단가 (VAT 별도) | 비고 |
|--------|------|-----------------|------|
| **구독 — 스타터** | 월 구독 | ₩29,000/월 | 전 강좌 스트리밍 + 진행률 |
| **구독 — 프리미엄** | 월 구독 | ₩49,000/월 | 스타터 + 코치 영상 피드백(월 2회) + Q&A |
| **단건 구매** | 코스별 구매 | ₩19,000~₩39,000/코스 | 소유권 영구 (구독 해지 후에도 유지) |
| **B2B 파트너십** | 수익 공유 | 스쿨 수익의 30% | Surftorial 플랫폼 제공, 스쿨은 콘텐츠 공급 |
| **기업/단체 위탁** | 연간 계약 | ₩500,000~/년 | 지자체·스포츠회관 대상, 맞춤 커리큘럼 |

### 수익 모델 전략

- **프리미엄(Freemium) 진입**: 무료 체험 강좌로 퍼널 진입 → 유료 전환
- **구독 LTV 극대화**: 스타터 시작 → 코치 피드백 체험 → 프리미엄 업셀
- **단건 구매는 전환 부스터**: 구독 부담 느낌 유저를 위한 진입점
- **B2B는 스케일 엔진**: 스쿨 네트워크 확보 시 유저 풀 급증

## 10. 시장 검증 가정 (Key Assumptions to Validate)

| # | 가정 | 검증 방법 |
|---|------|-----------|
| 1 | 한국 서핑 인구는 온라인 강좌에 지불 의향이 있다 | 랜딩 페이지 사전 등록 전환율 ≥ 5% |
| 2 | 오프라인 레슨 경험자가 온라인으로 전환한다 | 인터뷰, 설문 (현장 레슨 만족도·보완 니즈) |
| 3 | 코치 영상 피드백에 월 ₩20,000 프리미엄 지불한다 | A/B 테스트 (스타터 vs 프리미엄 전환율) |
| 4 | 서핑 스쿨이 수익 공유 모델에 참여한다 | 5개 스쿨 대상 파일럿 제안 |
| 5 | 비시즌(11~3월)에도 수요가 존재한다 | 비시즌 랜딩 트래픽·사전등록 측정 |

## 11. 1인 운영 현실 반영 메모

- **초기 콘텐츠**: 1~2개 코스(입문+중급)로 시작, 런칭 후 점진적 확대
- **코치 네트워크**: 초기 1~2명 인스트럭터와 전속 계약, 프리미엄 피드백은 코치가 직접 응답
- **마케팅**: 1인 운영 → 콘텐츠 마케팅(SEO) 중심, 퍼포먼스 광고는 최소
- **고객 지원**: 카카오톡 채널 1:1, FAQ/챗봇으로 자동화
- **B2B 확장**: 초기 1인 → 검증 후 순차적 스쿨 파트너십 확대
