Surftorial System Architecture

MVP Architecture — Next.js + Supabase + Cloudflare Stream

Frontend Layer

  • • Next.js 15 App Router (RSC)
  • • Tailwind CSS + shadcn/ui
  • • Zustand state management
  • • Vidstack video player (HLS)
  • • React Hook Form + Zod

Backend & Data Layer

  • • Next.js API Routes + Server Actions
  • • Prisma ORM over PostgreSQL 16
  • • Supabase Auth (OAuth + Email)
  • • Row Level Security policies
  • • Stripe Checkout + Webhooks

Infrastructure & Delivery

  • • Vercel (Edge deployment)
  • • Cloudflare Stream (VOD)
  • • Cloudflare DNS + CDN
  • • GitHub Actions CI/CD
  • • Sentry error monitoring

Surftorial MVP Architecture • v1.0.0 • Designed for 1-person development
